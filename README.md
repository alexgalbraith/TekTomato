# 🍅 TekTomato

A clean, minimal Pomodoro timer for Windows 10/11. Built as a personal productivity tool inspired by [YAPA 2](https://github.com/YetAnotherPomodoroApp/YAPA-2), with a focus on staying out of your way while giving you meaningful stats about your focus habits.

## Features

- **Always-on-top overlay** — floats above all your other windows, draggable anywhere on screen
- **Negative countdown** — when a Pomodoro finishes, the timer keeps running into negative numbers so you can wrap up naturally before acting
- **Nothing auto-starts** — every transition (break, next Pomodoro) requires a deliberate click
- **Adjustable opacity** — make the background transparent while keeping the countdown fully visible
- **Dark / Light / System Auto themes** — follows Windows theme automatically, or set it manually
- **Windows system sounds** — pick your completion sound from a dropdown of your installed Windows notification sounds
- **System tray** — minimise to tray, with a right-click menu to start/pause/exit
- **Stats: Calendar heatmap** — GitHub-style contribution graph showing Pomodoros per day
- **Stats: Line chart** — focus time over the last 3/6/12 months
- **Fully configurable** — work duration, break durations, long break interval, opacity, theme, sound

## Tech Stack

- **C# 12 / .NET 8** (WPF)
- **SQLite** via EF Core — local storage only, no cloud, no login
- **LiveCharts2** — line chart
- **SkiaSharp** — custom calendar heatmap rendering
- **H.NotifyIcon.Wpf** — system tray
- **CommunityToolkit.Mvvm** — MVVM pattern

## Building

### Prerequisites

- [Visual Studio 2022](https://visualstudio.microsoft.com/) (Community edition is fine)
  - Workload: **.NET desktop development**
- .NET 8 SDK (ships with VS 2022, or install from [dotnet.microsoft.com](https://dotnet.microsoft.com/download/dotnet/8.0))

### Steps

1. Clone the repo:
   ```
   git clone https://github.com/alexgalbraith/TekTomato.git
   ```
2. Open `TekTomato.sln` in Visual Studio 2022
3. Restore NuGet packages (VS does this automatically on first open)
4. Build → **Build Solution** (Ctrl+Shift+B)
5. Run with F5, or find the compiled `.exe` in `TekTomato/bin/Debug/net8.0-windows/`

> **Note:** The app stores data in `%AppData%\TekTomato\` — this is created automatically on first run.

## Configuration

All settings are accessible from the Settings panel (gear icon in the overlay). Changes take effect immediately.

| Setting | Default | Description |
|---------|---------|-------------|
| Work duration | 25 min | Length of a Pomodoro session |
| Short break | 5 min | Break after each Pomodoro |
| Long break | 15 min | Break after every N Pomodoros |
| Long break interval | 4 | Pomodoros between long breaks |
| Sound | Windows Notify System Generic.wav | Completion sound (from your Windows sounds) |
| Theme | System Auto | Dark, Light, or follows Windows |
| Background opacity | 80% | Transparency of the overlay window background |
